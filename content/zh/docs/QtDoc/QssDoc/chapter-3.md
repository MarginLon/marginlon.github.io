## 重置元素
&emsp;&emsp;重置所有内容有助于在不同操作系统之间统一风格
```css
* {
  padding: 0px;
  margin: 0px;
  border: 0px;
  border-style: none;
  border-image: none;
  outline: 0;
}
/* specific reset for elements inside QToolBar */
QToolBar * {
  margin: 0px;
  padding: 0px;
}
```
